<h1 align="center">Aspirantes ITVH · Proyecto Móvil</h1>

<p align="center">
  <strong>Aplicación Móvil Oficial Multiplataforma para el Proceso de Admisión</strong>
</p>

<p align="center">
  Aplicación móvil e infraestructura de frontend construida en <strong>Flutter</strong>, adaptada de manera modular del ecosistema principal de <strong>Comunidad ITVH</strong>. Su objetivo es ofrecer una red social, perfiles interactivos y herramientas de geolocalización interna para acompañar a los nuevos estudiantes en su ingreso al TecNM Campus Villahermosa.
</p>

### 🚀 Módulos del Sistema

<table>
  <tr>
    <td width="40" align="center">📱</td>
    <td><strong>Comunidad Aspirantes (Social Feed)</strong><br><sub>Red social interna nativa optimizada para que los aspirantes interactúen mediante publicaciones, comentarios y reacciones en tiempo real, respaldados por Supabase Realtime.</sub></td>
  </tr>
  <tr>
    <td align="center">👤</td>
    <td><strong>Gestión de Perfiles</strong><br><sub>Flujo completo de registro, autenticación segura y personalización de perfiles específicos para el proceso de admisión.</sub></td>
  </tr>
  <tr>
    <td align="center">🗺️</td>
    <td><strong>Módulo UbicaTecNM</strong><br><sub>Renderizado y mapeo interactivo de datos geográficos para guiar de forma dinámica a los nuevos alumnos dentro de la infraestructura del campus.</sub></td>
  </tr>
  <tr>
    <td align="center">📦</td>
    <td><strong>Almacenamiento Multimedia Extensivo</strong><br><sub>Integración con API compatible S3 a través de Cloudflare R2 para la carga, optimización y distribución eficiente de imágenes de perfil y contenido del feed social.</sub></td>
  </tr>
</table>
